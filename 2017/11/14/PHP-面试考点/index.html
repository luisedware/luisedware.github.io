实践真题

常量与数据类型

  • 回顾真题：PHP 的字符串可以使用哪三种定义方法以及各自的区别是什么？
  • 考官考点：PHP 的字符串的定义方式及各自区别

  • 知识延伸：常量及数据类型

  • 定义方式：

    • 单引号
    • 双引号
    • heredoc、newdoc
  • 区别：
    • 单引号不能解析变量
    • 单引号不能解析转义字符，只能解析单引号和反斜线本身
    • 双引号可以解析变量，变量可以使用特殊字符和 {} 包含
    • 双引号可以解析所有转移字符
    • 单双引号都可以使用 . 连接字符串或变量
    • 单引号效率高于双引号
    • newdoc 类似于单引号
    • heredoc 类似于双引号
    • heredoc、newdoc 比较擅长处理大文本

浮点类型不能运用到 == 或 === 比较运算中

布尔类型 false 的七种情况

  • 整型 0
  • 浮点 0.0
  • 布尔 false
  • 空字符串
  • 字符串 0
  • 空数组
  • null

常量

  • 定义
    • const
    • define
  • 区别
    • const 是语言结构，更快
    • define 是函数
    • const 不能用于类常量的定义，const 可以
  • 相同
    • 常量一经定义，不能被修改，不能被删除
  • 预定义常量
    • __LINE__：文件中的当前行号。
    • __FILE__：文件的完整路径和文件名。
    • __DIR__：文件所在的目录。
    • __FUNCTION__：返回函数被定义时的名字（区分大小写）。
    • __CLASS__：返回该类被定义时的名字（区分大小写）。
    • __TRAIT__：返回 trait 被定义时的名字（区分大小写）。
    • __METHOD__：返回该方法被定义时的名字（区分大小写）。
    • __NAMESPACE__：当前命名空间的名称（区分大小写）。

运算符考察点

  • 回顾真题：foo() 和 @foo() 之间的区别
  • 考官考点：PHP 运算符的错误抑制符
    • 延伸考点
      • 递增、递减运算符不影响布尔值
      • 递减 null 值没有效果
      • 递增 NULL 为 1
      • 逻辑运算符
        • 短路作用
          • $a = true || $b == 3
          • $b = false && $a == 1
        • || 和 && 与 or 和 and 的优先级不同
          • $a = false || true; || 优先 =，最后 $a = true;
          • $b = false or true; $b 为 false
          • $b = true or false; $b 为 true
  • 知识延伸：PHP 所有运算符考点

实践真题

  • 3 > 0 返回 true;造成判断条件短路
  • $a 等于 true
  • $a++ 等于 true
  • $b++ 等于 1
  • echo $a 显示 1，echo $b 显示 1

流程控制考察点

  • 回顾真题：请列出 3 种 PHP 数组循环操作的语法，并注明各种循环的区别
  • 考点分析：PHP 遍历数组的三种方法及各自区别
    • 方法
      • 使用 for 循环
      • 使用 foreach 循环
      • 使用 while、list()、each() 组合循环
    • 区别
      • for 只能遍历索引数组
      • foreach 可以遍历索引和关联数组，在遍历之前会自动对数组进行 reset() 操作
      • list()、each() 和 while 循环可以遍历索引和关联数组，但不会对数组指针进行 reset() 操作
  • 知识延伸：
    • 分支结构
      • if…else
      • if…elseif…else
        • 多个 elseif 从句是排斥关系
        • 使用 elseif 语句有一个基本原则，总把优先范围小的条件放在前面处理
      • switch…case
        • switch 后面的控制表达式的数据类型只能是整型、浮点类型或者字符串
        • continue 语句作用到 switch 的作用类似于 break
        • switch…case 会生成跳转表，直接跳转到对应 case，执行效率会快一点

自定义函数及内部函数

回顾真题

  • 考点分析
    • 变量的作用域和静态变量
      • 静态变量仅在局部函数域中存在，但当程序执行离开此作用域时，其值并不会消失
      • 静态变量的特点
        • 仅初始化一次
        • 初始化时需要赋值
        • 每次执行函数该值会保留
        • static 修饰的变量是局部的，仅在函数内部有效
        • 可以记录函数的调用次数，从而可以在某些条件下终止递归
  • 知识延伸
    • 函数的参数及参数的引用传递
    • 函数的返回值及引用返回
      • 函数值可以通过关键字 return 返回
      • 函数值可以返回 PHP 任意数据类型
      • 关键字 return 会中止函数执行，将控制权交回函数调用处
      • 省略关键字 return，返回值为 null，无法返回多个值
      • 从函数返回一个引用，必须在函数声明和指派返回值给一个变量时都是使用引用运算符 &
    • 外部文件的导入
      • 给出文件路径则按照文件路径查找
      • 没有文件路径则从 include_path 中查找
      • 如果 include_path 中没有，则从调用脚本所在的目录和当前工作目录下寻找
      • 加载过程中未找到文件，include 会发出一条警告，require 会发出一个错误。
    • 系统内置函数
      • 时间日期函数
        • date()
        • strtotime()
        • mktime()
        • time()
        • microtime()
        • date_default_timezone_set()
      • IP 处理函数
        • ip2long()
        • long2ip
      • 打印处理函数
        • print()
        • printf()
        • print_r()
        • echo
        • sprintf()
        • var_dump()
        • var_export()
      • 序列化、反序列化函数
        • serialize()
        • unserialize()
      • 字符串处理函数
        • implode()
        • explode()
        • join()

        • trim()
        • ltrim()

        • strstr()
