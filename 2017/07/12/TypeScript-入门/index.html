学习 TypeScript 的优势

  • 支持 ES6 规范
  • 强大的 IDE 支持：WebStorm
  • Angular 2~4 的开发语言

安装 TypeScript 的开发环境

  • 什么是 compiler?
    • 编译器
  • 为什么需要 compiler?
    • 需要将 TypeScript 转化为 JavaScript 运行
  • TypeScript 在线 compiler
    • TypeScript Playground
  • TypeScript 本地 compiler
    • npm install -g typescript
    • tsc –version
